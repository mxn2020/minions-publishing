---
title: Installation
description: Install Minions Publishing packages
---

## TypeScript / Node.js

```bash
npm install @minions-publishing/sdk minions-sdk
```

## Python

```bash
pip install minions-publishing
```

## CLI

```bash
npm install -g @minions-publishing/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
