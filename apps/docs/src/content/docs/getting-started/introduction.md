# Minions Publishing

Platform credentials, publish queue, and post receipts

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-publishing/sdk`) — Core library for Node.js
- **Python SDK** (`minions-publishing`) — Python bindings
- **CLI Tool** (`@minions-publishing/cli`) — Command-line interface
- **Web App** — Interactive playground at [publishing.minions.wtf](https://publishing.minions.wtf)
